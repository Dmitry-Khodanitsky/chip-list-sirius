### Chip List — управляемый список с попапом для скрытых элементов
Компонент для отображения набора чипов, который автоматически прячет не влезающие элементы в поповер. Адаптируется под изменение ширины экрана.

🔗 Демо: [https://chip-list-sirius.vercel.app/]

🛠 Запуск
`npm install`
`npm run dev`

🛠 Сборка:
`npm run build`
`npm run preview`

#### 🚀 Технологии
React 19, TypeScript, Vite, @radix-ui(поповер)

#### Ключевые директории:
- `src/components/Chip` — отдельный переиспользуемый компонент чипа
- `src/components/ChipList` — логика списка чипов и хук `useVisibleChips`
- `src/components/Popover` — вспомогательный поповер для скрытых чипов
- `src/utils/chipCalculations.ts` — утилиты для вычисления доступного места и видимых чипов

#### 🧠 Ключевые решения
- **Render prop:** в компоненте `ChipList` использован паттерн render prop. `ChipList` перпринимает в качестве пропса функцию `renderItem`, которая получает элемент (`Chip`) и возвращает JSX. Это даёт гибкость, можно заменить визуализацию чипа без изменения внутренней логики расположения и управления переполнением. 
Преимущества:
	- Разделение ответственности: `ChipList` контейнер - отвечает за видимость и расположение элементов внутри, а `renderItem` — за отображение каждого элемента.
	- Возможность использования различных типов чипов (с иконками, Action-кнопками и т.д.) без изменения кода контейнера.
  - ChipList инкапсулирует логику позиционирования и калькуляции, не навязывая конкретную реализацию UI.

- Хук `useVisibleChips`: вычисляет, какие чипы помещаются в доступной ширине, используя измерения и логику из `chipCalculations`. Вынесение в хук упрощает поддержку и делает логику повторно используемой.

- Утилита `chipCalculations.ts`: содержит функции для расчёта ширины элементов и определения количечтва видимых чипов.
